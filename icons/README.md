# Icons

This directory should contain the extension icons in the following sizes:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For now, you can use placeholder images or create simple icons with the BoongAI branding.

To generate icons quickly, you can use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Or create them manually using design tools like Figma, Canva, or GIMP.
